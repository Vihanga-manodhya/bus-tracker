# 🚌 Bus Fleet Tracker (Pilot Project)

A real-time GPS tracking and fleet management system built with Flutter. This pilot project is designed to track bus locations, calculate average speeds between stops, and help identify reasons for schedule delays. 

Currently, this repository focuses on the **Flutter Web** interface (Admin Dashboard) for real-time monitoring.

---

## 🚀 Features (MVP)
* **Real-time GPS Tracking:** Monitors live coordinates of the bus fleet.
* **Passive Tracking:** Runs silently without distracting the driver.
* **Web Dashboard:** A live map interface for depot managers and timekeepers.
* **Firebase Integration:** Instant data syncing using Cloud Firestore.
